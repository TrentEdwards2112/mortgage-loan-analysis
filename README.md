# Mortgage Loan Analysis

This project analyzes mortgage loan data using SQL (MySQL) and Tableau to identify patterns in loan approvals, loan amounts, and loan volume across states.

The goal of the analysis is to evaluate how loan size may influence approval outcomes and uncover trends in regional lending patterns.

---

## 🧠 Background

This project is informed by my 8+ years of experience in mortgage operations, including loan processing, underwriting support, appraisal review, and reporting.

Through this experience, I developed a strong understanding of how loan size, borrower profiles, and risk factors influence approval decisions.

This analysis applies those domain insights using SQL and Tableau to explore approval trends across states.

---

## 🎯 Objective

To analyze mortgage loan data and identify relationships between loan size and approval rates across states, using SQL for data preparation and Tableau for visualization.

---

## 🔧 Tools Used

- MySQL (data cleaning and analysis)
- Tableau (data visualization and dashboard development)

---

## 📁 Project Overview

1. Cleaned and standardized raw mortgage data:
   - Loan_Status
   - Employment_Status
   - State

2. Built SQL queries to analyze:
   - Approval rates by state
   - Average loan amount by state
   - Total loan volume by state

3. Developed a Tableau dashboard to visualize key metrics and support analysis

---

## 📊 Key Metrics Analyzed

- Approval Rate by State
- Average Loan Amount by State
- Total Loan Volume by State

## 📈 Key Insight

States with higher average loan amounts (e.g., CA, NY) tend to have lower approval rates, while states with lower loan amounts (e.g., OH, IL) show higher approval rates—suggesting an inverse relationship between loan size and approval likelihood.
